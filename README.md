# 📘 CFG Debugger with CNF Conversion & CYK Simulation

## 🚀 Overview

This project is a web-based application that allows users to:

* Input a Context-Free Grammar (CFG)
* Convert it into Chomsky Normal Form (CNF)
* Simulate the CYK algorithm step-by-step
* Visualize parse trees and state diagrams

---

## ✨ Features

* ✅ CFG parsing
* ✅ ε-production removal
* ✅ Unit production elimination
* ✅ CNF conversion
* ✅ CYK algorithm simulation
* ✅ Parse tree generation
* ✅ State diagram visualization
* ✅ Step-by-step explanation

---

## 🧠 Concepts Used

* Context-Free Grammar (CFG)
* Chomsky Normal Form (CNF)
* CYK Algorithm
* Dynamic Programming

---

## 🛠️ Tech Stack

* Python (Flask)
* HTML, CSS, JavaScript
